Comparison Protocol for Encoded Guidelines

Using Guideline Interchange Format
(GLIF-2)

I. For each encoding of the guideline, determine the following:

  1. Name of guideline

  2. Number of times each type of construct was used

    • Interfaces:

      • Guideline_Model
      • Guideline_Step (abstract class)
      • Action_Step
      • Conditional_Step
      • Branch_Step
      • Synchronization_Step
      • Action_Specification
      • Patient_Data
      • Criterion
      • Boolean_Criterion
      • K_Of_N_Criteria
      • Supplemental_Material
      • Local_Material
      • WWW_Material

    • Enumerated Types:

      • t_ordering
        • any_order
        • parallel
      • t_selection
        • all_of
        • some_of
        • one_of
      • t_continue
        • wait_for_all
        • proceed_after_one
        • loop
      • t_logic_type
        • k_two_valued
        • k_three_valued

  3. List of patient data elements

  4. name
  5. type
  6. possible values
  7. temporal constraints

  8. Intention of guideline

  9. Eligibility criteria of guideline

  10. Subguidelines used

  11. Level of detail covered

  12. Additional detail (knowledge) added

  13. Other comments about authoring patterns noted

II. Compare the two encoded versions of the guideline.
  1. Authoring institution of each encoding
    • Guideline encoding number "1" = (give name of authoring institution)
    • Guideline encoding number "2" = (give name of authoring institution)

  2. Compare number of times each type of construct was used
    • Which constructs are used by both?
    • Which constructs are used by one but not the other?
    • Which constructs are used by neither?
    • How many action steps or conditional steps do the two encodings have in common?

  3. Compare list of patient data elements
    • Encoding number 1:
      • How many data elements in encoding number 1?
      • How many of those data elements are also shared by encoding number 2?
      • What percent are shared?
      • How many of those data elements are not shared by encoding number 2?
      • What percent are not shared?
    • Encoding number 2:
      • How many data elements in encoding number 2?
      • How many of those data elements are also shared by encoding number1?
      • What percent are also shared?
      • How many of those data elements are not shared by encoding number 1?
      • What percent are not shared?

  4. Compare intention of guideline
    • Which encodings specify an intention?
      • Both
      • One only
      • Neither
    • If both specify an intention, how do they compare in meaning?
      • Same
      • Similar but not exactly the same
      • Different

  5. Compare eligibility criteria of guideline
    • Which encodings specify eligibility criteria?
      • Both
      • One only
      • Neither
    • How do the eligibility criteria in the 2 encodings compare?
      • Same
      • Similar
      • Different

  6. Compare number of subguidelines used
    • Which encodings use subguidelines?
      • Both
      • One only
      • Neither
    • If both encodings use subguidelines, are any of the subguidelines the same?
      • Yes
      • No

  7. Compare level of detail covered
    • How do the encodings compare in terms of level of detail covered?
      • Same level of detail
      • One has a little more detail than the other
      • One has much more detail than the other

  8. Compare additional detail (knowledge) added
    • Which encodings added more knowledge not explicitly given in the guideline source text?
      • Both
      • One only
      • Neither

  9. Compare other comments about authoring patterns noted
    • Summarize differences.
