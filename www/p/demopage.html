Vocabulary server: A demonstration of the use of the server with the Stanford T-HELPER system.

Guideline server concept: MGH demo of guideline tool.

DXplain as a server: Brief MGH demo of their own web-interface to DXplain.

Columbia systems: A demonstration of vocabulary server use, guideline server use, including calls to DXplain with the clinical information system at Columbia.

DXplain Interface: Detmer demo of his interface to DXplain (our third example of a tool for using DXplain as a server).

WebMedline: Detmer/Shareck demo of shared network-based effort to link WebMedine with BWH's medical journal content server/resources.

Application environment: Decision Systems Group at Brigham and Women's Hospital shows their application environment which integrates clinical data, forms, images, web content and guideline trees.
