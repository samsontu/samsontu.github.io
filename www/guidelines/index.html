Stanford Guidelines Representation Work  (up to 1996)

  • Samson's AMIA Fall Conference paper. 
  • Max Borten's breast mass guideline represented in GLIF: John Gennari's interpretation (ref: email of July 2)
  • BreastMass.glif: The output of Samson's Protege-to-Glif translator, given "BreastMass.pins" as input. This is John Gennari's version of the Breast Mass Workup Guideline in GLIF2.
  • BreastMass.zip: A Protege/Win ontology, KA-tool and instance file for the Max Borten Breast Mass Workup guideline. You must have Protege/Win (version 1.1) installed to make much sense of these files.
  • Breast Cancer Treatment Protocol EST 2190 in GLIF
  • Breast Cancer Treatment Protocol EST 2190 in GLIF2 (see email of August 26)
  • BreastCancer.zip: Protege/Win file for Protocol EST 2190 and associated ontology and KA tool files. Need Protege/Win (version 1.1) to use these files.
  • Flu Guideline, encoded by Larry Fagan using Protege/Win 1.1.
  • flu.zip: Protege/Win file for the flu guideline and associated ontology and KA tool files. Need Protege/Win (version 1.1) to use these files.
  • Partial Encoding of Encephalopathy Guideline, encoded by Samson Tu using Protege/Win 1.1.
  • Samson Tu's proposed execution model for GLIF2
